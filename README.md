# Product Catalog API

## Description
The **Product Catalog API** is a Spring Boot application designed to manage a product catalog. It provides RESTful endpoints for managing products and integrates with a PostgreSQL database for data persistence.

## Features
- REST API for product management
- PostgreSQL database integration
- Spring Security for authentication and authorization
- Validation for request payloads
- Gateway support with Spring Cloud Gateway

## Technologies
- **Java 21**
- **Spring Boot 3.5.7**
- **Spring Data JPA**
- **Spring Security**
- **Spring Cloud Gateway**
- **PostgreSQL**
- **Maven**

## Requirements
- **Java 21** or higher
- **Maven** for dependency management
- **PostgreSQL** database
